Fichier HTML

Le premier fichier index.html divisé en deux sections :

La première section regroupe le menu et les offres spéciales en promotions.

  1. utilisation de <div> pour regrouper les éléments du menu.
  2. L'élément <fieldset> pour regrouper les éléments d'un produit et mettre des légendes.
  3. A chaque div, on associe une classe pour pouvoir modifier dans la feuille de style.
  4. <s> pour barrer du texte

La deuxième section regroupe la barre de recherche, de navigation, le produit du jour, les messages du site, et les différents produits.

  1. Accueil contient un lien vers la page d'accueil.
  2. Panier contient un sous-menu des différentes actions.
  3. Produit du jour (Galaxy Fold) avec un lien pour le bouton Acheter. REMARQUE : tous les liens des articles renvoient vers ce produit.
  4. Une fois le produit selectionné, le bouton Ajouter au panier renvoie vers un autre fichier HTML avec un bouton Retirez du panier, celui-ci renvoie vers la page d'accueil.

Une dernière partie pour le pied de page qui contient des informations , liens vers les documents presentation.htmlet détails.html.

Feuille de style CSS

  • Selecteur universel * pour selectionner tous les éléments HTML, on met margin et padding à 0 pour ne plus avoir d'espace.
  • Pour les sections, on fixe display : flex, tous les enfants seront de type flex, donc n'est plus considéré comme un élément inline ou bloc. Le z-index permet de configurer un ordre, ainsi lorsque deux élements se chevauchent, le z-index le plus élévé sera au dessus.
  • Pour les différents éléments, on fixe les dimensions avec : width, height, padding, margin.
  • font-family pour la police,color et background-color pour les couleurs, text-align: center pour centrer le texte.
  • Menu afficher en block avec display : block.
  • :hover permet de spécifier l'apparence d'un élément au moment où l'utilisateur le survole avec le pointeur, sans nécessairement l'activer.
  • cursor: pointer, pour changer le pointeur de la souris au passage sur l'élément.
  • display: inline-block, pour créer des blocks qui s'adaptent à la largeur du navigateur, plus facile à utiliser que float. Inline-block permet de mettre les éléments en ligne tout en conservant les caractéristiques du block, on peut modifier la hauteur, largeur ...
  • position: absolute les propriétés de position de l'élément se fait par rapport à son ancêtre.
  • position: relative, les propriétés de position de l'élément se font directement sur l'élément, top/bottom pour le décalge vertical et left/right pour le décalage horizontale.
  • x:nth-child(n) La règle s'applique à chaque x lorsqu'il est le n ème enfant de son parent.
  • Fiche technique du produit, sous forme de tableau défini en HTML.
  • p.ma_classe {...} La règle s'applique à tous les éléments div de classe ma_classe.
  • Responsive CSS : on définit une largeur maximal de 1028px, en dessous de cette largeur, les propriétés que l'on a définit prennent la place des propriétés initiales.
